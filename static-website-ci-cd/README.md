# 🌐 Static Website CI/CD on AWS

## 📌 Overview

This project demonstrates deploying a static website using AWS S3 with automated CI/CD using GitHub Actions.

## 🏗️ Architecture

GitHub → GitHub Actions → AWS S3 → Static Website

## ⚙️ Services Used

* Amazon S3
* IAM
* GitHub Actions

## 🚀 Features

* Automated deployment on push
* Secure credential handling using GitHub Secrets
* Scalable static hosting
